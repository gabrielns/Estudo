# XR Studio Manager - Banco de Dados
Projeto de banco de dados relacional para gestão de um estúdio de Realidade Virtual e Aumentada. 
O sistema gerencia projetos, tarefas, inventário de hardware (VR Headsets) e licenças de software.

1. Abra seu gerenciador de banco de dados (MySQL Workbench ou pgAdmin).
2. Crie um novo database: "CREATE DATABASE xr_manager;".
3. Execute o script "criar_tabelas.sql" para gerar a estrutura.
4. Execute o script "inserts.sql" para carregar os dados.
5. Utilize os demais scripts para testar as funcionalidades.
